> Контейнер, выдающий зависимости по имени/типу; клиент не знает, откуда они берутся.

## Где помогает
- Быстрый прототип без явного DI.
- Плагины/модули, которые требуют реестр сервисов.

## Мини-пример
```ts
const locator = new Map([["logger", console]]);
function useLogger(){ locator.get("logger").log("hi"); }
```

## Замечания
- Прячет зависимости, усложняет тестирование — чаще лучше явный DI.
- Можно использовать как переходный этап при рефакторинге.
