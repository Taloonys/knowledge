> Глобальная таблица сервисов/настроек по ключу.

## Где помогает
- Простая точка доступа к общим зависимостям в маленьких проектах/скриптах.
- Подмена реализаций в рантайме.

## Мини-пример
```python
registry={"logger": print}
def get(name): return registry[name]
get("logger")("hi")
```

## Замечания
- Легко скатиться в скрытые зависимости; предпочитайте явные аргументы.
- Подходит для конфигураций/фабрик, но избегайте разрастания.
