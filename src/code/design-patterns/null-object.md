> Вместо `null` подставляем безопасную реализацию интерфейса.

## Где помогает
- Убирает проверки `if (obj)` при работе с сервисом.
- Тесты/заглушки: логгер без вывода, пустая метрика.

## Мини-пример
```js
const NullLogger = { log(){ /* no-op */ } };
const logger = maybeLogger ?? NullLogger;
logger.log("ok");
```

## Замечания
- Убедитесь, что нулевая реализация корректна для домена (не скрывает ошибки).
- Для сложных объектов полезно прокидывать метки/счётчики вызовов.
