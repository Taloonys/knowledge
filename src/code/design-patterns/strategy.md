> Алгоритм инкапсулирован в отдельный объект и подменяется на лету.

## Где помогает
- Разные способы оплаты, сортировки, сериализации.
- Выбор алгоритма по конфигу/окружению.

## Мини-пример
```python
def pay(amount, strategy):
    strategy(amount)

pay(100, lambda a: print(f"cash {a}"))
pay(100, lambda a: print(f"card {a}"))
```

## Замечания
- Стратегии должны иметь общий интерфейс и быть взаимозаменяемыми.
- Можно сочетать с фабрикой для выбора стратегии.
