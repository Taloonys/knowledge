> Сущность-Атрибут-Значение: хранение произвольных свойств в key-value структуре.

## Где помогает
- Набор полей заранее неизвестен или сильно варьируется.
- Нужна гибкость без миграций на каждое новое поле.

## Мини-пример
```ts
type Attrs = Record<string, Record<string,string>>;
const attrs: Attrs = { "42": { color:"red", size:"L" } };
const size = attrs["42"]["size"]; // "L"
```

## Замечания
- Цена гибкости: сложнее индексация, валидация и запросы.
- Используйте ограниченно: аудит/метаданные/редкие поля.
