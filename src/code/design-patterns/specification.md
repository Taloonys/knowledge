> Логические предикаты, которые можно комбинировать (`and/or/not`) и повторно использовать.

## Где помогает
- Фильтры/правила доступа/валидации.
- Перенос условий из сервисов в отдельный слой.

## Мини-пример
```js
const adult = u => u.age >= 18;
const hasDocs = u => !!u.passport;
const canEnter = u => adult(u) && hasDocs(u);
```

## Замечания
- Хорошо работает с репозиториями: спецификация -> SQL/ORM фильтр.
- Комбинации спецификаций легко кешировать и тестировать отдельно.
