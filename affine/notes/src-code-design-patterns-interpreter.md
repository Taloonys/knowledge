> Маленький язык с парсером/деревом и простым вычислителем.

## Где помогает
- Мини-DSL для фильтров/правил/шаблонов.
- Быстро описать и исполнить выражения без тяжёлого парсера.

## Мини-пример
```python
def eval_expr(tokens):
    left,op,right = tokens
    if op == "+": return left+right
    if op == "-": return left-right
    raise ValueError("op")

eval_expr([1,"+",2])  # 3
```

## Замечания
- Для роста языка лучше перейти на готовые парсер-комбинаторы.
- Держите грамматику маленькой и тестируйте edge cases.
