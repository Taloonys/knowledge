# ASCII

–ü–µ—Ä–≤—ã–º –ø–æ—è–≤–∏–≤—à–∏–º—Å—è —Å—Ç–∞–Ω–¥–∞—Ä—Ç–æ–º –∫–æ–¥–∏—Ä–æ–≤–∫–∏ –±—ã–ª **ASCII7** (**American standard code for information interchange 7 bits**). –ö–æ—Ç–æ—Ä—ã–π –≤–º–µ—â–∞–ª –≤–µ—Å—å –Ω–µ–æ–±—Ö–æ–¥–∏–º—ã–π –Ω–∞–±–æ—Ä —Ç–æ–ª—å–∫–æ –ª–∏—à—å —Å–∏–º–≤–æ–ª–æ–≤ –ª–∞—Ç–∏–Ω–∏—Ü—ã –∏ –Ω–µ–∫–æ—Ç–æ—Ä—ã—Ö —Å–ª—É–∂–µ–±–Ω—ã—Ö –≤ 7 –±–∏—Ç - 128 –∑–Ω–∞—á–µ–Ω–∏–π, –≥–¥–µ –ø–µ—Ä–≤—ã–µ 31 –≤—ã–¥–µ–ª—è–ª–∏—Å—å –ø–æ–¥ —É–ø—Ä–∞–≤–ª—è—é—â–∏–µ —Å–∏–º–≤–æ–ª—ã.

# ANSI

–î–ª—è —Ä–µ—à–µ–Ω–∏—è –≤–æ–ø—Ä–æ—Å–∞ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–∏–∑–∞—Ü–∏–∏ –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã—Ö —Å–∏–º–≤–æ–ª–æ–≤ –¥–ª—è –∫–∞–∂–¥–æ–π —Å—Ç—Ä–∞–Ω—ã –±—ã–ª –ø—Ä–∏–¥—É–º–∞–Ω —Å—Ç–∞–Ω–¥–∞—Ä—Ç **ANSI** (**American National Standards Institute**). –ì–¥–µ –¥–æ–±–∞–≤–ª—è–µ—Ç—Å—è —Ç–∞–∫–∞—è –≤–µ—â—å, –∫–∞–∫ **–∫–æ–¥–æ–≤–∞—è —Å—Ç—Ä–∞–Ω–∏—Ü–∞**, –∫–æ—Ç–æ—Ä–∞—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–ª–∞ —Å–æ–±–æ–π —É–∫–∞–∑–∞–Ω–∏–µ, –∫–∞–∫–æ–≥–æ —Ä–æ–¥–∞ —Å–∏–º–≤–æ–ª—ã (–∫–∞–∫–æ–π —Å—Ç—Ä–∞–Ω—ã, –Ω–∞–ø—Ä–∏–º–µ—Ä) –±—É–¥—É—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –ø—Ä–∏ –∫–æ–¥–∏—Ä–æ–≤–∫–µ. –û–¥–Ω–∞–∫–æ –ø—Ä–æ–±–ª–µ–º—É —ç—Ç–æ –æ—Å–æ–±–æ –Ω–µ —Ä–µ—à–∏–ª–æ, —Ç.–∫. –≤–æ—Å—Ç–æ—á–Ω—ã–µ —Å—Ç—Ä–∞–Ω—ã –∏–º–µ—é—Ç –±–æ–ª–µ–µ —Ç—ã—Å—è—á–∏ –∏–µ—Ä–æ–≥–ª–∏—Ñ–æ–≤, –∫–æ–≥–¥–∞ –≤—Å—ë –µ—â—ë –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è —Ä–∞—Å—à–∏—Ä–µ–Ω–Ω–∞—è –º–æ–¥–µ–ª—å ASCII —Å –ª–∏—à—å 256 –∑–Ω–∞—á–µ–Ω–∏—è–º–∏. –¢–æ–≥–¥–∞ –¥–æ–±–∞–≤–∏–ª–∏ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å —Ö—Ä–∞–Ω–µ–Ω–∏—è –Ω–µ–∫–æ—Ç–æ—Ä—ã—Ö —Å–∏–º–≤–æ–ª–æ–≤ –ø–æ –∫–æ–¥—É —Å—Ç—Ä–∞–Ω—ã –≤ 2 –±–∞–π—Ç–∞—Ö (—É–∂–µ 65536 –∑–Ω–∞—á–µ–Ω–∏–π), –æ–¥–Ω–∞–∫–æ –∫–∞–∂–¥—ã–π –ø—Ä–æ–¥–æ–ª–∂–∞–ª –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —É–∂–µ —Å–≤–æ–∏ —Å—Ñ–æ–º—Ä–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–µ —Å–∏–º–≤–æ–ª—ã, –ø—Ä–æ–±–ª–µ–º—ã —Å—Ç–∞–Ω–¥–∞—Ä—Ç–∞ —ç—Ç–æ –Ω–µ —Ä–µ—à–∏–ª–æ.

# Unicode

–ë—ã–ª–∞ —Å–æ–∑–¥–∞–Ω–∞ –¥–∏–Ω–∞–º–∏—á–µ—Å–∫–∏ —Ä–∞—Å—à–∏—Ä—è—é—â–∞—è—Å—è —Ç–∞–±–ª–∏—Ü–∞ –∫–æ–¥–∏—Ä–æ–≤–∫–∏ **UNICODE**, –≥–¥–µ —Å–∏–º–≤–æ–ª –º–∏–Ω–∏–º—É–º –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–ª—Å—è 2 –±–∞–π—Ç–∞–º–∏, –ø–µ—Ä–≤—ã–µ 128, –∫–∞–∫ –æ–±—ã—á–Ω–æ, —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–æ–≤–∞–ª–∏ ASCII, –æ–¥–Ω–∞–∫–æ –ø–æ—Å–ª–µ 2 –±–∞–π—Ç –∏–º–µ–ª–∞—Å—å –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –¥–æ–±–∞–≤–ª—è—Ç—å –Ω–æ–≤—ã–µ —Å–∏–º–≤–æ–ª—ã. –¢.–µ. –ø–æ—è–≤–∏–ª–∏—Å—å –∏ –≤–µ—Ä—Å–∏–∏ UNICODE.
