> Разделяем общее неизменяемое состояние между множеством мелких объектов.

## Где помогает
- Много однотипных сущностей: символы текста, тайлы, частицы.
- Нужно экономить память.

## Мини-пример
```js
const pool=new Map();
const glyph = ch => pool.get(ch) || pool.set(ch,{ch}).get(ch);
const a1=glyph("a"), a2=glyph("a");
console.log(a1===a2); // true
```

## Замечания
- Делите состояние на внутреннее (shared) и внешнее (уникальное).
- Следите за очищением пула, чтобы не держать лишние ссылки.
