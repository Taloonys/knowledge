# Токенизация
> Как в компиляции - происходит разбиение строки на лексемы/токены
# Парсинг
> Синтаксический анализ, где на основе токенов формируется дерево AST, и производится синтаксическая проверка
# Семантический анализ
> Где производится смысловые зависимости, типы данных и видимость/доступность объектов.
# Интерпретация
> Код-строчка преобразуется в байт код (не машинный код!, это промежуточный и понятный виртуалке код), который потом исполняется виртуальной машиной
